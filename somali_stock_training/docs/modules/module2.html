Overview

Fisheries stock assessment relies on diverse data types. This module explores what data is available, how it’s collected, and how it’s used.

Learning Objectives

By the end of this module, participants will:

Data Types Covered

1. Catch Data

  • Landings by species
  • Temporal trends
  • Spatial information
  • Data sources and quality

2. Effort Data

  • Fishing effort metrics
  • Fleet characteristics
  • Seasonality
  • Regional variations

3. Catch Composition Data

  • Species composition
  • Size composition
  • Discard data
  • Bycatch information

4. Biological Sampling Data

  • Length measurements
  • Age data
  • Maturity observations
  • Condition factors

5. Environmental Data

  • Sea surface temperature
  • Oceanographic conditions
  • Seasonal patterns
  • Climate impacts

Somali Fisheries Context

Typical Data Available

  • Landing site monitoring (partial)
  • Market surveys (limited)
  • Observer programs (developing)
  • Research surveys (sparse)

Gaps and Challenges

  • Limited historical data
  • Spatial and temporal gaps
  • Quality inconsistencies
  • Limited biological sampling

Data Quality Framework

Data Quality Assessment:
├── Completeness (% of data collected)
├── Accuracy (measurement error)
├── Consistency (within and across regions)
├── Timeliness (how current is the data)
└── Validity (plausibility of values)
