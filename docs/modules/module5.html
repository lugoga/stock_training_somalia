## Overview

Statistical analysis transforms raw data into insights for stock assessment and management. This module introduces practical analytical approaches.

## Learning Objectives

By the end of this module, participants will:

- [ ] Apply descriptive statistics to fisheries data
- [ ] Conduct time series analysis
- [ ] Calculate stock indicators
- [ ] Create effective visualizations
- [ ] Interpret results and communicate findings

## Descriptive Statistics

### Summary Metrics
- Mean, median, standard deviation
- Distribution of catch
- Size composition trends
- Species diversity indices

### By-Group Analysis

- Spatial patterns (by region, site)
- Gear comparisons
- Species dynamics
